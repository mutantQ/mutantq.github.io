# 1a. 뇌를 모방한 프랙탈 구조의 메모리-컴퓨터

폰 노이만 구조에서와 같이 연산과 기억이 분리된 구조는 뇌의 실제 구조와는 거리가 멀다. 오히려, 뇌는 가능한 모든 간선 중 일부만을 가지는 희소 네트워크(Sparse Network)로 모델링하는 것이 자연스럽다[[ELEC0](https://www.notion.so/ELEC0-Covers-the-topic-of-neuromorphic-computing-as-a-replacement-of-Von-Neumann-systems-d06a9ded39e34a2eb6cd654ed027470c?pvs=21)]. 또한, 통상적인 반도체의 그리드 배치가 아닌, 중심부로부터 방사형으로 뻗어나오는 형태의 구조를 가지고 있다. 구체적으로는, 척추를 통해 뇌의 중심부로 전달된 신호는 백색질의 축삭 돌기를 통해 방사형으로 뻗어나와 회색질의 세포체에 다다른다[[BRAIN0](https://www.notion.so/BRAIN0-The-brain-has-radial-symmetry-72b23ef34bad4b7ea2593caa583ed949?pvs=21)].

연관된 데이터일수록 인접한 영역에 저장하는 프랙탈 구조의 인메모리 컴퓨터를 제안한다.

위의 구조는 CNN 등 인접한 정보 사이에서만 단계적인 연산이 필요한 작업에 최적일 수 있다. 저장하는 정보는 반드시 이진수(0 또는 1)일 필요가 없으며, 실수를 저장해도 무방하다. 또한, 연산 과정에서 정확하고 오차 없는 연산을 기대하지는 않는다. 인간의 뇌 역시 그렇다.

공정이 쉽다. 만일 적합한 물질 조합으로부터 결정 구조를 키워내서 메모리 구조가 자발적으로 생성되도록 할 수 있으면, 고가의 EUV로 메모리 반도체를 찍어내는 것보다 훨씬 저렴할 가능성이 존재한다.

사람은 모든 정보를 정확하게 처리하는 대신 정보를 적절하게 압축하여 유사한 개념을 인접한 구역에 저장하고 다시 그 개념을 상황에 맞추어 적용하는 것에 능하다.

검색 키워드: Neuromorphic computing, fractal memory

찾은 추가 키워드: VLSI, analog computing, oxide-based memristors, spintronic memories, fractal connectivity, multivariate LSTM.
